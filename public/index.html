Read the below instructions to add, get, update, or delete users in the system database:

For PUT add a query string with an attribute/value pair to the request body:

  • Example:
  • /create/username=billsmith&password=pw123

For GET add a query string with an attribute/value pair to Retrieve

  • Example:
  • /retrieve?username=wpchu

For UPDATE, use POST with the following body example form:

  • /update/find={"username":"wpchu"}&update={"$set":{"password":"new_pw123"}}
  • however, this won't be used that much in my final project b/c users will not be able to update their information

For DELETE, add a query to find the user you want to delete:

  • Example:
  • /delete/find={"username":"wpchu"}
